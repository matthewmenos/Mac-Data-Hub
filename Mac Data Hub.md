Mac Data Hub



Features:

1. All payments are in Ghana Cedis and pesewas
2. Use Paystack as Payment Method.
3. Supported Networks: MTN, Telecel and Airtel Tigo.
4. PWA
5. Reseller Dashboard.
6. All data bundle sold on the site will be powered by a source with API.
7. Guests can buy and checkout without an an account.
8. All bundles will be done for Ghanain munbers.
9. An admin dashboard to control all the prices.
10. Users can apply to be a reseller as a fee. (All configurable in the admin dashboard).
11. Users can create their own store inside the web app and this is how it works:
* User create a storefront
* Set prices above the base prices provided by the admin.
* Earn profit when someone use the link to buy the data.
* The profit is calculated as Storefront Owner's - Admin prices.

12\. Deposit and withdrawals all powered by Paystack.

13\. No simulation.
14. HTML, CSS and vanilla JS for frontend

15\. Python Flask for backend.

The web app will use API keys from Gigzhub.net to deliver data bundles.

Humanize every content.
Database will be SQLITE dual database system.( Global db and One User, One database).
